Functions

Functions can only be written in JavaScript and called upon in reaction to an event. Allow the user to enter a color. When they click the button (event), that will call a function that will display a message and user input.

Water Weight

Assume a gallon of water weighs 8.33lbs. Allow a user to enter a number of gallons. Upon clicking a button, call a function to calculate the total weight and display that value.

Loops

Single Milestone

Multiple Milestones
